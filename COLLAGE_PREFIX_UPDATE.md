# Collage Prefix Implementation - Update Summary

## ✅ What Was Added

You now have a **prefix system to identify auto-generated collage images**, making it easy to find, manage, and regenerate collages in the future.

## 🏷️ The Prefix

```
AUTO_COLLAGE_
```

**Example filenames:**
- `AUTO_COLLAGE_collage_1712957400000.png` ← Auto-generated by system
- `user_vacation_photo.jpg` ← User-uploaded photo
- `AUTO_COLLAGE_collage_1712957401000.png` ← Auto-generated by system

## 📝 Changes Made

### 1. Updated `/utils/collageGenerator.js`
- ✅ Added `COLLAGE_PREFIX` constant export: `'AUTO_COLLAGE_'`
- ✅ Updated `blobToFile()` to automatically prepend prefix to filename
- ✅ Prevents duplicate prefixes if already present

**Code:**
```javascript
export const COLLAGE_PREFIX = 'AUTO_COLLAGE_';

export const blobToFile = (blob, fileName = 'collage.png') => {
    const prefixedFileName = fileName.startsWith(COLLAGE_PREFIX) 
        ? fileName 
        : `${COLLAGE_PREFIX}${fileName}`;
    return new File([blob], prefixedFileName, { type: blob.type });
};
```

### 2. No changes needed to `/pages/createpost.js`
The prefix is automatically applied when `blobToFile()` is called.

## 🎯 Use Cases Now Possible

### ✅ 1. Find All Collages
```javascript
import { COLLAGE_PREFIX } from '../utils/collageGenerator';

const collages = files.filter(f => f.startsWith(COLLAGE_PREFIX));
// Result: ['AUTO_COLLAGE_collage_1234.png', 'AUTO_COLLAGE_collage_5678.png']
```

### ✅ 2. Regenerate All Collages
```javascript

const collages = allFiles.filter(f => f.startsWith(COLLAGE_PREFIX));

// Delete old collages
for (const collage of collages) {
    await deleteFile(collage);
}

// Regenerate with new format/quality
// ... regeneration code ...
```

### ✅ 3. Find Posts with Collages
```javascript
const postsWithCollages = posts.filter(post =>
    post.postPhotos?.[0]?.includes(COLLAGE_PREFIX)
);
```

### ✅ 4. Batch Analytics
```javascript
const stats = {
    totalImages: files.length,
    autoCollages: files.filter(f => f.startsWith(COLLAGE_PREFIX)).length,
    userPhotos: files.filter(f => !f.startsWith(COLLAGE_PREFIX)).length,
};
```

### ✅ 5. Prevent Duplicate Processing
```javascript
if (filename.startsWith(COLLAGE_PREFIX)) {
    // Skip - this is already a collage
    continue;
}
```
